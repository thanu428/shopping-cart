# 🛒 Shopping Cart Application

A Java-based application that simulates an online shopping cart system and demonstrates an automated testing pipeline using DevOps tools like Git, Maven, and Jenkins.

---

## 🌟 Features

* 📦 Add products to the shopping cart
* 🗑 Remove products from the cart
* 📋 View cart items
* 💰 Calculate total cart price
* 🧪 Automated unit testing using JUnit
* ⚙ Continuous integration using Jenkins

---

## 🛠 Technologies Used

* **Java**
* **Apache Maven**
* **Git & GitHub**
* **Jenkins**
* **JUnit**

---

## ▶️ How to Run

### 1️⃣ Clone the Repository

```
git clone https://github.com/thanu428/shopping-cart.git
```

### 2️⃣ Go to Project Folder

```
cd shopping-cart
```

### 3️⃣ Compile the Project

```
mvn compile
```

### 4️⃣ Run Tests

```
mvn test
```

### 5️⃣ Run the Application

```
mvn exec:java
```
